# ThunderTools

Panel administrativo para servidores privados de WoW (estilo Battle.net), con:
- Splash + login administrativo (primera ejecución pide credenciales y las guarda).
- Módulos visibles sin conexión; acciones deshabilitadas en offline.
- Ajustes para emulador, expansión, SOAP y DB (repack 5).
- CI en GitHub Actions que construye y firma el APK automáticamente.
